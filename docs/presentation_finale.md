# Presentation finale - AcadReminder

## 1. Introduction

AcadReminder est une application web academique developpee avec Django pour aider les etudiants a suivre les dates importantes.

## 2. Probleme

Les etudiants oublient souvent les inscriptions, examens, soutenances, depots de dossiers et paiements administratifs.

## 3. Solution proposee

La plateforme centralise les evenements academiques, affiche des notifications et fournit un chatbot capable de repondre aux questions courantes.

## 4. Acteurs

- Etudiant : consulte les evenements, configure les rappels et utilise le chatbot.
- Administrateur : gere les evenements, les utilisateurs et supervise le systeme.

## 5. Fonctionnalites principales

- Authentification.
- Calendrier academique.
- Chatbot.
- Notifications et preferences.
- Tableau de bord etudiant.
- Supervision administrateur.
- Interface administrateur dediee pour ajouter, modifier et supprimer les evenements.
- Integration Gemini optionnelle pour enrichir les reponses du chatbot.

## 6. Technologies

- Django.
- SQLite en developpement.
- HTML5, CSS3 et Bootstrap 5.

## 7. Demonstration

1. Creer un compte etudiant.
2. Consulter le tableau de bord.
3. Filtrer le calendrier academique.
4. Poser une question au chatbot.
5. Modifier les preferences de rappel.
6. Se connecter comme administrateur et ajouter un evenement depuis l'interface de gestion.
7. Superviser le systeme.

## 8. Conclusion

AcadReminder contribue a reduire les oublis academiques et ameliore l'organisation des etudiants grace aux rappels et au chatbot integre.
