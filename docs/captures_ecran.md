# Captures d'ecran a preparer

Ce document sert de checklist pour les captures demandees dans les livrables.

## Captures recommandees

- Page d'accueil.
- Page d'inscription etudiant.
- Page de connexion.
- Tableau de bord etudiant.
- Calendrier academique avec filtres.
- Detail d'un evenement.
- Chatbot avec une question sur les examens.
- Page des notifications.
- Page des preferences de rappel.
- Page de supervision administrateur.
- Django Admin : liste des evenements.

## Conseil

Lancer le serveur avec `python manage.py runserver`, ouvrir chaque page dans le navigateur, puis enregistrer les captures dans ce dossier ou dans un dossier `docs/screenshots/`.
