# Guide utilisateur - AcadReminder

## Etudiant

1. Creer un compte depuis la page d'inscription.
2. Se connecter.
3. Consulter le tableau de bord pour voir les prochaines dates.
4. Ouvrir le calendrier pour rechercher ou filtrer les evenements.
5. Exporter un evenement ou tout le calendrier au format `.ics` pour l'ajouter a son agenda personnel.
6. Utiliser le chatbot pour poser une question, par exemple :
   - Quand commencent les examens ?
   - Quelle est la date limite d'inscription ?
   - Quand aura lieu la soutenance ?
   - Quelles sont les prochaines dates importantes ?
7. Configurer les preferences de rappel.
8. Consulter les notifications internes et les marquer comme lues.

## Administrateur

1. Se connecter a `/admin/`.
2. Ajouter ou modifier les evenements academiques.
3. Consulter les profils, preferences, notifications et messages du chatbot.
